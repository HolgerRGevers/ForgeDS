# ForgeDS Fonts

This directory holds the Geist variable font files used by the ForgeDS IDE.

## Expected files

- `Geist-Variable.ttf` -- variable sans-serif (weight 100-900)
- `GeistMono-Variable.ttf` -- variable monospace (weight 100-900)
- `OFL.txt` -- SIL Open Font License

## Upstream

Sourced from [vercel/geist-font](https://github.com/vercel/geist-font).
If filenames change in a future release, update the `@font-face` declarations in `web/src/index.css`.
